# MovieGo - Interactive Prototype Creation

## Project Overview

MovieGo is an interactive movie ticket booking app prototype designed in Figma. The project demonstrates how users can navigate through an application, select a movie, book tickets, and receive booking confirmation through a clickable prototype.

## Features

* Interactive navigation between screens
* Clickable buttons and user flows
* Smart Animate transitions
* Mobile-friendly interface
* User testing and feedback analysis

## Screens Included

1. Welcome Screen
2. Home Screen
3. Movie Details Screen
4. Booking Confirmation Screen

## Prototype Flow

Welcome Screen → Home Screen → Movie Details Screen → Confirmation Screen

## User Testing

Quick user testing was conducted to evaluate navigation and usability. Feedback was collected and minor design improvements were made to enhance the user experience.

## Tools Used

* Figma
* GitHub

## Outcome

This project helped in understanding interactive prototyping, navigation flows, screen transitions, user testing, and design refinement.
