
# User Testing Report

## Participants

3 Users

## Tasks Performed

1. Open the application.
2. Select a movie.
3. Book a ticket.
4. Return to the home screen.

## Feedback

* Navigation was intuitive.
* Booking flow was easy to follow.
* Buttons were clearly visible.
* Transitions improved the user experience.

## Improvements Made

* Increased button sizes.
* Improved spacing between UI elements.
* Added smooth Smart Animate transitions.
